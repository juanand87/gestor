# Gestor

Sistema de gestión desarrollado con PHP.

## Descripción

Este proyecto es un sistema de gestión web.

## Requisitos

- PHP 7.4 o superior
- MySQL 5.7 o superior
- Apache (XAMPP)

## Instalación

1. Clona este repositorio en tu directorio htdocs de XAMPP
2. Importa la base de datos (si aplica)
3. Configura los parámetros de conexión
4. Accede desde tu navegador: `http://localhost/gestor`

## Estructura del Proyecto

```
gestor/
├── index.php          # Punto de entrada principal
├── config/            # Archivos de configuración
├── includes/          # Funciones y clases PHP
├── assets/            # CSS, JS, imágenes
│   ├── css/
│   ├── js/
│   └── img/
└── views/             # Vistas/plantillas
```
